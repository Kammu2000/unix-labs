# Unix Labs

Recreating classic Unix tools for fun, learning, and understanding how computers work underneath the abstractions.

This repository is a playground for exploring:

- Computer Architecture
- Operating Systems
- Concurrency
- File Systems
- Performance Engineering
- CLI Design

Tools are implemented in different languages to compare ideas, trade-offs, and language ecosystems.

## Tools

- cat 
- head
- tail 
- wc
- grep 
- find
- du

More coming soon...

## Philosophy

Build → Benchmark → Optimize → Learn

The goal is not feature parity with production tools, but to understand the concepts and engineering behind them.
